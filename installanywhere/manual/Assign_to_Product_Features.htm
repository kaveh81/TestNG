Assign to Product Features

InstallAnywhere 8.0

For each feature that has been created in the Features subtask, there is a column added to the Assign To Product Features view of the Visual Tree. For each file, folder, or action displayed there will be a box in the column for each feature.

To assign a file, folder, or action to a feature, put a check in the box.

If a folder has a check in the box it means that all the items in the folder belong to that feature. If there is a dash (-) in the box it means that some items, but not all items of the folder are selected to belong to that feature. If no items belong to a feature the box will be blank.

Files or actions will either have a check if it is associated with a feature or blank if it is not associated with a feature.
