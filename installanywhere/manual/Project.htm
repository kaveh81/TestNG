Project

InstallAnywhere 8.0

The Project task displays InstallAnywhere project information, configures, provides options to bundle or define acceptable VMs, and defines product, file installation, and localization options. The Project task includes the following subtasks:

  • Info
  • Description
  • File Settings
  • Platforms
  • Locales
  • Rules
  • Config
  • Java
