Registration

InstallAnywhere 8.0

When evaluating InstallAnywhere, the trial versions of InstallAnywhere are fully functional and allow developers to utilize all of the product's features. The installers that are built with an unregistered trial version of InstallAnywhere expire after three days. Purchasing a license and registering a copy of InstallAnywhere enables developers to build permanent installers.

Note

Projects built with trial versions of InstallAnywhere need to be rebuilt after registering. Simply close and re-open InstallAnywhere, and click Build.

With a license and a serial number, developers may register and activate InstallAnywhere online. The activation process sends the IP address, serial number, and registration information over the Internet. No other information is sent to Macrovision. If access to the Internet is unavailable, this information may be saved to a file and emailed to the Macrovision support team.

  • Macrovision promises to keep registration information private and Macrovision will never sell or distribute it outside of Macrovision. Macrovision uses this information solely to ensure the integrity of Macrovision's registered user database.
  • It is a violation of the license agreement to use multiple copies of InstallAnywhere with the same serial number.
  • If there are any questions about the InstallAnywhere registration and validation processes, please contact the Macrovision support team.

If InstallAnywhere is installed behind a firewall, the registration may time out. If it does it will create a text file that can be emailed to the Macrovision support team for registration. For registered InstallAnywhere customers using a version prior to 6.0, the older serial number will not activate version 6.0. Please contact the Macrovision sales department for upgrade information.

Task

To register InstallAnywhere:

  1. Open InstallAnywhere and click Preferences on the initial splash screen or click Register on the Help menu.
  2. Select the Registration tab.
  3. Type in the serial number and other information and click Register.
  4. Close and re-open InstallAnywhere.
