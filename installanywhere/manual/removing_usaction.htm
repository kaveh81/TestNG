Removing an Enable Update Notifications Action

InstallAnywhere 8.0

You can also remove an action that prompts the installation to integrate with FLEXnet Connect using the Install task.

Task

To remove an Enable Update Notifications action:

  1. On the Advanced Designer, click Install.
  2. On the Visual Tree, click the Enable Update Notifications action you want to remove.
  3. Click Remove. The Enable Update Notifications action no longer appears in the Visual Tree.
