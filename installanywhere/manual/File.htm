File

InstallAnywhere 8.0

The File menu contains the standard Open and Save commands with a few more options.

New

New enables the creation of a new InstallAnywhere development project. To open and name a new project, click Save As within the Create New Project dialog box. The Create New Project dialog will also allow developers to open templates by selecting Open Other.

Open

Open will display the Open Project File dialog. Along with text field for Folder

Open Recent

Open Recent contains a pop-up list of recently opened InstallAnywhere project files.

Save

Save saves the project. Since the project is named upon creation, this task will never request a name.

Save As

Save As allows developers to rename the project. The current project still exists, but remains in the state it was at it's last save. The renamed project contains any changes and will the current open project.

Save a Copy As

Save a Copy As allows developers to rename the project, but unlike Save As, Save a Copy As will only make a copy, leaving the original project as the current open project.

Revert to Saved

Revert to Saved, reverts to the last saved state of the project. If developers want to save a version of the project before reverting, Select Save a Copy As, then Revert to Saved.

Exit

Exit will close the current project and exit the InstallAnywhere development environment. If any changes have been made since launching or the last save, there will be a prompt from the Exit.
