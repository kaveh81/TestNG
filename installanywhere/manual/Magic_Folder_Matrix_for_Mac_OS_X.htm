Note

Items in italics cannot be written to by the typical user. An asterisk (*) equals all mounted volumes (partitions) except Startup.

Using an authenticated installer allows the installer to be run as the Root user. Mac OS X Choose Folder Panels have aliases to such locations as Desktop or Home. End-users selecting these from a native Mac OS X dialog box will have then have them set to the Root user's Desktop, Home, etc. Consequently, an end-user may not be able to access these files installed to these locations. We suggest that in such cases the developer should parse the return paths using the Match Regular Expression Rule and alert the end-users to this special case.
