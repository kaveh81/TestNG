Assign to Components

InstallAnywhere 8.0

The Assign To Components view of the Install task works somewhat differently from the Assign To Product Features view. File check boxes are circular radio buttons and folder check boxes are square. Since files may only belong to one component, if there are three Components — Comp1, Comp 2, and Comp 3 — selecting Comp 1 for a file means it cannot belong to any of the other Components.

Components are assigned automatically up when adding the files, but may be modified. If all files within a folder are assigned to the same component, the folder will have a check. A folder will be assigned to multiple components if files within that folder are assigned to different components. If items within a folder are assigned to multiple components the box will contain a dash (-). If no items in the folder belong to the component, the box will be blank.
