Visual Tree

InstallAnywhere 8.0

The Visual Tree displays the hierarchy of files, folders, and actions that make up your install.

To add files to the Visual Tree click Add Files. In addition, Files and folders can be dragged and dropped directly from Windows Explorer, the Mac OS X finder, and some Unix/Linux File Managers directly into the file tree.

Items in the Visual Tree may be dragged and dropped to rearrange them hierarchically. The up, down, right and left arrows may be used to move the file. The right arrow will move the item into the folder listed above it. The left arrow will move the item out of the folder to a higher level in the hierarchy.

Rules may be assigned to any item in the hierarchy. All rules associated with a file, folder, or action will be evaluated before the item it is associated with will be installed.

Right-clicking will bring up a context sensitive menu that will add, duplicate or clear actions or add files. Right-clicking also enables developers to specify a magic folder where they want to place the item.
