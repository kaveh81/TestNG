Bundling Java Runtime Environments

InstallAnywhere 8.0

Since InstallAnywhere installers are Java based, they require a Java Virtual Machine (VM) to run. InstallAnywhere installers can bundle a virtual machine with the installer, thus eliminating the possibility that the installer will not find a suitable virtual machine for the product or for the installer.

Developers can choose whether or not the end-user will have a choice of VMs to use for the installed product.

InstallAnywhere installers run from the bundled virtual machine. In order to bundle a virtual machine, it must be packaged as a VM Pack. Free VM packs may be found on the Macrovision Web site (details on how to create your own VM Packs can be found below).

A VM will not be installed under the following conditions:

  • The installer is built without an included VM.
  • The installer is built with an included VM, but the end-user opts to use a VM already on their system by choosing the appropriate option on the Choose Virtual Machine installer action.
  • LaunchAnywhere executables weren't installed (including the Uninstaller's LaunchAnywhere executable file).

A VM will be installed under the following conditions:

  • The installer is built with an included VM, and the end-user opts to install the VM by choosing the appropriate option on the Choose Virtual Machine installer action.
  • The installer is built with an included VM, and the Choose Virtual Machine action has been disabled via the Pre-Install task in the Advanced Designer. (The one exception to this rule is if no LaunchAnywhere applications are being installer or uninstallers being created. In that situation, a VM isn't needed and won't be installed.)
  • The bundled VM in the Build task of the Advanced Designer is selected.
  • If an end-user deletes the VM that was last used by LaunchAnywhere, LaunchAnywhere will search for a new VM that matches the setting in the lax.nl.valid.vm.list LaunchAnywhere property. Installers deployed to non-Latin systems require an international Java Virtual Machine.
