Getting Started

InstallAnywhere 8.0

InstallAnywhere has two authoring modes. The Project Wizard makes the process easier by making choices for developers. The Advanced Designer gives developers greater control of an installer project.

InstallAnywhere opens displaying the first frame of the Project Wizard unless the default preference has been changed using the Preferences command in the Edit menu. To access an existing InstallAnywhere project, click Open Existing Project. Click the project name in the Open Recent Project list then click Advanced Designer.

The general process for developing an InstallAnywhere project is:

  1. Create New Project
  2. Set Project Information
  3. Add Pre-Install Actions
  4. Install Tasks
  5. Add Post-install Actions
  6. Set Installer UI Options
  7. Configure the Project Uninstaller
  8. Build
  9. Test
