Using Help

InstallAnywhere 8.0

The InstallAnywhere online help contains these sections:

Getting Started

The Getting Started section includes tutorials and other introductory information about InstallAnywhere. The five minute introductory tutorial walks through the Project Wizard. The advanced tutorial walks through procedures which illustrate the power and flexibility of the Advanced Designer. Every user should take the time necessary to go through this chapter. This section also describes the two editions of InstallAnywhere. This section also includes instructions on how to register InstallAnywhere, and how to convert existing InstallAnywhere projects.

Key Concepts

The Key Concepts section briefly addresses some of the key concepts you should be aware of when creating installers with InstallAnywhere. InstallAnywhere is an extremely strong and flexible tool, and these descriptions give you the conceptual framework to understand the available options. Many of the key concepts introduced in this section have related sections devoted to them exploring the subjects in greater detail.

Common Solutions

The Common Solutions section addresses ways of handling situations that typically arise when building installers. If you are not sure how to achieve a specific goal, this is a good place to start looking for answers.

The remaining reference sections discuss fundamentals and options of InstallAnywhere in detail.

Web-Based Online Help

Web-based online help is available to you 24 hours a day, seven days a week, on our Web site at http://helpnet.macrovision.com.
