General Settings

InstallAnywhere 8.0

Use the General Settings tab to select default settings for InstallAnywhere.

InstallAnywhere Setup

This setting determines whether or not the InstallAnywhere Project Wizard opens when you start InstallAnywhere.

  • Click Project Wizard if you want the Project Wizard to appear when you start InstallAnywhere.
  • Click Advanced Designer if you want the Advanced Designer to appear when you start InstallAnywhere.

Project Loading

This setting determines whether InstallAnywhere should verify the location of files referenced in the Install task when a project opens.

  • Click Always (recommended) if you want InstallAnywhere to verify the location of files referenced in the Install task when a project opens.
  • Click Never if you want to allow a project to open without InstallAnywhere verifying the location of files referenced in the Install task.

Command Line Builds

This setting determines whether a project built from the command line can successfully build if referenced files are missing.

  • Click Fail (recommended) if you want a project built from the command line to fail if referenced files are missing.
  • Click Continue without including the missing files to allow a project built from the command line to continue building with referenced files that are missing.

Manifest Files

InstallAnywhere uses manifest files to identify files that are included in the installer. Manifest files are useful when groups of developers are working on a project in separate locations.

This setting determines whether a project that includes a manifest file can successfully build if files referenced in the manifest file are missing.

  • Click Fail (recommended) if you want a project that includes a manifest file to fail if files referenced in the manifest file are missing.
  • Click Continue without including the missing files to allow a project that includes a manifest file to continue building with files referenced in the manifest file that are missing.

Default Install Task View

This setting determines whether the default view for the Install task displays Actions assigned to Components or Product Features.

  • Click Components if you want the Install task to display Actions assigned to Components by default.
  • Click Product Features if you want the Install task to display Actions assigned to Product Features by default.

Designer Language

This setting determines the language of the text in the InstallAnywhere interface. To change the language, click the language in the list. You must close and re-open InstallAnywhere for this change to take effect.
