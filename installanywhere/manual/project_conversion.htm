Project Conversion

InstallAnywhere 8.0

Projects built with InstallAnywhere 4.5 and earlier will be converted to the new InstallAnywhere 7 format. When trying to load an older project into the new Advanced Designer, you will be notified that the original project will be saved. Converted projects may have components generated automatically. Automatically generated components will have the following note:

AUTOGENERATED COMMENT: The following comment is an autogenerated aggregation of all the Feature descriptions from which this component was generated during project conversion.

For more information on Install Sets, Features, and Components, see Install Sets.
