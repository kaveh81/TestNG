Build Tools

InstallAnywhere 8.0

A build properties file template named Build.properties can be found inside your InstallAnywhere application folder, inside resource/build.

Build.properties Arguments 

Argument

Description

-v

Print the InstallAnywhere product versionC:\Program Files\InstallAnywhere 7 Enterprise>build.exe -v InstallAnywhere 7 Enterprise

-p

Use specified build properties file. More information on the build.properties can be found in the next segment of this document

+

Add platform to build

-

Remove platform from build

The following arguments modify the add and remove platforms arguments:
