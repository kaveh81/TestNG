Customizing an Enable Update Notifications Action

InstallAnywhere 8.0

The Enable Update Notifications customizer contains the following tabs:

  • General Settings tab
  • Advanced Settings tab

Important

You must register your product with FLEXnet Connect before you can successfully integrate it with your installation project. If you have not already registered your product, click Click here on the General Settings tab. A FLEXnet Connect window appears. Enter your username and password and click Sign In. Register your product using the screen prompts. Click the quick help icons for more information.

Task

To customize the General Settings tab:

  1. Click the Enable Update Notifications action you want to customize on the Virtual Tree. The Enable Update Notifications customizer appears.
  2. Click the General Settings tab.
  3. On the Path list, click the location you want the FLEXnet Connect files to be installed. A magic folder for the chosen destination with the Enable Update Notifications action appears in the Visual Tree.

    4. Note

    By default, the Enable Update Notifications customizer is configured to create a LaunchAnywhere executable. If you do not want to include a LaunchAnywhere executable for this action, clear the Create LaunchAnywhere to execute update check check box. All remaining settings on the General Settings tab are disabled.

  4. In the Launcher Name box, type a description for this launcher.

    5. Note

    You can change the graphical icon for this launcher like any LaunchAnywhere executable. You can also create a link, alias, or shortcut for this launcher. See Advanced Tutorial - Using the Advanced Designer for more information on using these settings in the Advanced Designer.

Task

To customize the Advanced Settings tab:

  1. Click the Enable Update Notifications action you want to customize on the Virtual Tree. The Enable Update Notifications customizer appears.
  2. Click the Advanced Settings tab.
  3. By default, FLEXnet Connect messages appear in the same language as the installer runtime. To select another language, click the Custom radio button and click the desired language on the list.

    4. Important

    The following languages supported in InstallAnywhere are not available for FLEXnet Connect: Czech, Catalan, Icelandic. If you do not specify a custom language in this case, the FLEXnet Connect language defaults to English.

  4. By default, InstallAnywhere registers your product with FLEXnet Connect using the version of your InstallAnywhere project. To designate an alternative version of FLEXnet Connect to be included with your product, click the Custom radio button and specify the version in the box.
  5. Specify the amount of days between the times that FLEXnet Connect checks for notifications for your product in the Update check interval (days) box.

    6. Note

    This setting only applies to installation authors using API calls (AppUpdate and AutoUpdate) in the agent.

  6. By default, InstallAnywhere uses the server hosted by Macrovision to be in communication with FLEXnet Connect. To designate another server, click the Self-hosted URL radio button and enter the URL in the box.
  7. By default, InstallAnywhere uses the FLEXnet Connect library that is included. If you want to specify another version of the FLEXnet Connect library, click the Custom location radio button and either enter the .jar filename in the box or click Choose File to browse to the file.
