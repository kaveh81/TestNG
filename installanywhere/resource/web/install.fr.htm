Instructions pour Windows :

    Instructions

    • Après le téléchargement, double-cliquez sur Install.exe


    Remarques :

    • Vous n’avez pas besoin d’installer d’autres logiciels. Une VM Java est incluse dans ce téléchargement.
    • Vous devrez peut-être installer la VM Java 1.5 ou ultérieure. Vous pouvez télécharger la version requise depuis le site Web de Sun.
    • Si la VM Java n’est pas installée sur votre système, téléchargez le paquetage ci-dessus.

Instructions pour Mac OS X :

    Instructions

    • Après le téléchargement, double cliquez sur Installer


    Remarques :

    • SE minimum requis : Mac OS X 10.4
    • Votre système doit disposer de la VM Java 1.5 ou ultérieure. Vous pouvez télécharger la version requise depuis le site Web de Apple.
    • Le fichier compressé de l’installateur est normalement reconnu et décompressé automatiquement par Stuffit Expander. Si ce n’est pas le cas, vous pouvez le décompresser manuellement avec StuffIt Expander 6.0 ou une version ultérieure.
    • Si vous n’arrivez pas à démarrer l’installateur après sa décompression, vérifiez que vous avez bien utilisé Stuffit Expander. Si les problèmes persistent, contactez le support technique.


    (Début)

Instructions pour toutes les autres plates-formes :

    Instructions (Unix ou Unix en tant que systèmes d’exploitation)

    • Pour Java 2, après le téléchargement, tapez
      java -jar install.jar
    • Pour Java 1.1, après le téléchargement, tapez
      jre -cp install.jar install
    • En cas d’échec, tapez
       java -classpath [chemin vers]classes.zip:install.jar install
    • Si le problème persiste, sous un shell de type sh, tapez
       cd [chemin de install.jar téléchargé]   CLASSPATH=install.jar
        export CLASSPATH
        java install
    • Sous un shell de type csh, tapez
        cd [chemin de install.jar téléchargé]
        setenv CLASSPATH install.jar
        java install


    Instructions (pour les autres plates-formes)

    • Votre système doit disposer de la VM Java 1.5 ou ultérieure. Vous pouvez télécharger la version requise depuis le site Web de Sun.
    • Dans une fenêtre console, passez au répertoire de téléchargement de  install.jar avant d’exécuter l’installateur
    • Votre système d’exploitation peut invoquer Java de différentes façons. Pour démarrer l’installateur, ajoutez install.jar à votre variable CLASSPATH et démarrez la classe main de l’installateur nommée install
