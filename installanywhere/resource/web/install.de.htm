Windows-Anleitung:

    Anleitung

    • Doppelklicken Sie nach dem Herunterladen auf Install.exe.


    Hinweise

    • Sie müssen keine weitere Software installieren. In diesem Download ist eine Java Virtual Machine enthalten.
    • Sie müssen möglicherweise eine Java Virtual Machine Version 1.5 (oder höher) installieren. Sie können diese von der Java-Website von Sun herunterladen.
    • Wenn Sie noch keine Java Virtual Machine installiert haben, laden Sie hier das Paket herunter, das eine solche enthält.

Mac OS X-Anleitung:

    Anleitung

    • Doppelklicken Sie nach dem Herunterladen auf Installieren


    Hinweise

    • Erfordert Mac OS X 10.4 oder höher
    • Sie müssen Java 1.5 oder höher installiert haben. Sie können Java von der Apples-Website herunterladen.
    • Der komprimierte Installer sollte vom Dienstprogramm StuffIt Expander erkannt und nach dem Herunterladen automatisch dekomprimiert werden. Ist dies nicht der Fall, dekomprimieren Sie ihn manuell mit dem Programm StuffIt Expander 6.0 oder höher.
    • Wenn Sie Probleme beim Starten des dekomprimierten Installers haben, überzeugen Sie sich, dass der komprimierte Installer mit StuffIt Expander dekomprimiert wurde. Wenn weiterhin Probleme auftreten, wenden Sie sich an den technischen Support.

AIX-Anleitung:

    Anleitung

    • Öffnen Sie nach dem Download eine Shell und wechseln Sie mit  cd in das Verzeichnis, in das Sie den Installer heruntergeladen haben.
    • Geben Sie bei der Eingabeaufforderung Folgendes ein:  sh ./Install.bin


    Hinweise

    • Sie müssen eine Java Virtual Machine Version 1.5 (oder höher) installieren. Sie können diese von der Java-Website von Sun herunterladen oder bei dem Hersteller Ihres Betriebssystems anfordern.
    • In diesem Download ist eine Java Virtual Machine enthalten. Sie wird automatisch ausgeführt, wenn Sie das Shell-Skript starten.
    • Wenn Sie noch keine Java Virtual Machine installiert haben, laden Sie hier das Paket herunter, das eine solche enthält. Ansonsten können Sie die Java-VM auch von der Java-Website von Sun herunterladen oder bei dem Hersteller Ihres Betriebssystems anfordern.

Anleitungen für alle anderen Plattformen:

    Anleitung für UNIX oder UNIX-ähnliche Betriebssysteme

    • Geben Sie bei Java 2 nach dem Herunterladen Folgendes ein:
      java -jar install.jar
    • Geben Sie für Java 1.1 nach dem Herunterladen Folgendes ein:
      jre -cp install.jar install
    • Wenn dies nicht funktioniert, versuchen Sie es mit
       java -classpath [Pfad zu]classes.zip:install.jar install
    • Wenn dies auch nicht zu dem gewünschten Ergebnis führt, versuchen Sie es bei sh-Shells mit
       cd [zum Verzeichnis, in dem install.jar liegt]
        CLASSPATH=install.jar
        export CLASSPATH
        java install
    • Oder versuchen Sie es bei csh-Shells mit
        cd [zum Verzeichnis, in dem install.jar liegt]
        setenv CLASSPATH install.jar
        java install


    Anleitung für andere Plattformen

    • Sie müssen Java 1.5 oder höher installiert haben. Sie können Java von der Sun-Website herunterladen.
    • Wechseln Sie in einem Konsolenfenster vor dem Ausführen des Installers in das Verzeichnis, in das Sie  install.jar heruntergeladen haben.
    • Ihr Betriebssystem ruft Java möglicherweise auf andere Weise auf. Um den Installer zu starten, fügen Sie  install.jar zu Ihrem CLASSPATH hinzu und starten dann die Hauptklasse des Installers mit dem Namen install
