Windows での手順:

    手順

    • ダウンロードが完了したら、Install.exe をダブルクリックします。


    注意

    • その他のソフトウェアをインストールする必要はありません。このダウンロードには、Java VM も含まれています。
    • Java 1.5 (またはそれ以降) のVM をインストールしなければならない場合があります。Java VM は、Sun のJava Web サイト からダウンロードできます。
    • お使いのコンピュータに Java VM がインストールされていない場合は、上記パッケージの中から、VM 付きのものを選択してダウンロードしてください。

Mac OS X での手順:

    手順

    • ダウンロードが完了したら、Install をダブルクリックします。


    注意

    • Mac OS X 10.4 以降が必要です。
    • Java 1.5 以降がインストールされていることを確認します。Java は Apple のサイトからダウンロードできます。
    • 圧縮されたインストーラは、ダウンロード後に Stuffit Expander によって認識され、自動的に解凍されます。自動的に解凍されない場合は、StuffIt Expander 6.0 以降 を使用して、手動で解凍できます。
    • 解凍後、インストーラを適切に起動できない場合は、圧縮されたインストーラを Stuffit Expander 以外のアプリケーションで解凍しなかったかどうかを確かめてください。それでも問題が解決しない場合は、テクニカル サポートまでご連絡ください。

AIX での手順:

    手順

    • ダウンロードが完了したら、シェルを開き、 cd コマンドを使用してインストーラをダウンロードしたディレクトリに移動します。
    • プロンプトで次のように入力します: sh ./Install.bin


    注意

    • Java 1.5 (またはそれ以降) のVM をインストールする必要があります。Java VM については、Sun の Java Web サイトからダウンロードするか、またはお使いの OS の製造元にお問い合わせください。
    • このダウンロードには Java VM が含まれており、シェル スクリプトの実行時に自動的に実行されます。
    • お使いのコンピュータに Java VM がインストールされていない場合は、上記パッケージの中から、VM 付きのものを選択してダウンロードしてください。それ以外の場合は、Sun の Java Web サイトから VMをダウンロードするか、お使いの OS の製造元に問い合わせる必要があります。

その他のすべてのプラットフォームでの手順:

    手順 (Unix または Unix 系オペレーティング システム)

    • Java 2 の場合は、ダウンロード完了後に次のように入力します。
      java -jar install.jar
    • Java 1.1 の場合は、ダウンロード完了後に次のように入力します。
      jre -cp install.jar install
    • 上記の方法で適切にインストールできない場合は、次のように入力してみてください。
       java -classpath [該当するパス]classes.zip:install.jar install
    • それでも適切にインストールできない場合は、sh 系のシェルで次のように入力してください。
       cd [install.jar があるディレクトリ]
        CLASSPATH=install.jar
        export CLASSPATH
        java install
    • または、csh 系のシェルの場合は、次のように入力してください。
        cd [install.jar があるディレクトリ]
        setenv CLASSPATH install.jar
        java install


    手順 (その他のプラットフォーム)

    • Java 1.5 以降がインストールされていることを確認します。Java は Sun のサイトからダウンロードできます。
    • インストーラを実行する前に、コンソール ウィンドウでinstall.jar をダウンロードしたディレクトリに移動します。
    • オペレーティング システムによっては、異なる方法で Java を呼び出すものもあります。インストーラを起動するには、まず install.jar を CLASSPATH に追加し、その後で install という名前のインストーラのメイン クラスを起動します。
