Introduction

InstallAnywhere est la solution de choix adoptée par les développeurs d’installations multiplate-forme du monde entier. Combinaison exclusive d’ergonomie et de puissance, InstallAnywhere permet de créer très rapidement des installations multiplate-forme. Pour mieux répondre à vos besoins, InstallAnywhere existe en versions Standard Edition et Enterprise Edition. Avec InstallAnywhere, vous pouvez envisager les tâches suivantes :

  • Créer un projet d’installateur supportant plusieurs plates-formes et plusieurs langues.
  • Déployer des logiciels sur toute plate-forme client ou serveur.
  • En suivant les conseils de l’Assistant de projet, builder un installateur multiplate-forme en quelques minutes.
  • Exécuter les fonctionnalités des plates-formes cibles dont vous avez besoin pour simplifier le déploiement des logiciels.

Environnement minimum requis

Systèmes disposant d’InstallAnywhere (Authoring Environment)

  • 128 Mo de RAM
  • Affichage 8 bits (256 couleurs)
  • Résolution 1024 x 768

InstallAnywhere supporte les versions les plus récentes des systèmes d’exploitation mentionnés ci-après (avec application éventuelle des mises à jour, des correctifs/patchs et des Service Packs les plus récents) :

  • Windows Server 2003, XP, 2000, NT
  • Mac OS X, Mac OS X Server
  • Solaris/Sparc
  • Linux/Intel (Red Hat, SuSE)
  • HP-UX
  • AIX (5.2 ou version ultérieure)

Remarque – La version anglaise et les versions localisées (français, allemand, japonais) d’InstallAnywhere Enterprise Edition sont disponibles uniquement sur la plate-forme Windows. Les installateurs peuvent être buildés sur la plate-forme de votre choix – quelles que soient la plate-forme et la langue cibles. InstallAnywhere Enterprise Edition permet de builder des installateurs dans 29 langues.

Systèmes cibles de l’installateur (Installer Environment)

  • 64 Mo de RAM
  • Affichage 8 bits (256 couleurs)
  • Résolution d’écran 640 x 480

Les installateurs peuvent être exécutés sur toutes les versions des systèmes d’exploitation ci-dessous (support de Java obligatoire) :

  • Windows Server XP, 2003 (processeurs Itanium 2, x86, AMD-64 ou équivalent)
  • Windows Vista, 2000, NT, Me, 98
  • Linux/Intel (Red Hat, Red Hat Enterprise Server, UnitedLinux, TurboLinux, SuSE, etc.)
  • Linux (processeurs PowerPC, AMD-64, Itanium 2 ou équivalent)
  • Mac OS X (processeurs PowerPC, Intel ou équivalent)
  • Solaris Sparc, AMD-64, Intel
  • HP-UX (processeurs Itanium 2, PA-RISC ou équivalent)
  • Tru64, FreeBSD
  • AIX
  • z/OS
  • NetWare
  • Autres plates-formes UNIX

Remarque – Les installateurs InstallAnywhere ne sont pas supportés par les versions bêta ni par les préversions (« access release »).

VM Java supportées

  • Sun : 1.3.x, 1.4.x, 1.5.x, 1.6.x
  • IBM : 1.3.x, 1.4.x, 1.5.x
  • Apple : 1.3.x, 1.4.x, 1.5.x
  • HP : 1.3.x, 1.4.x, 1.5.x

Par défaut, l’installateur InstallAnywhere est associé à Java 1.5 sur toutes les plates-formes. Toute VM Java peut être intégrée à un installateur (« bundled ») pour garantir que le système cible répondra aux exigences minimales des installateurs et de vos applications. Pour télécharger les Packs VM disponibles, consultez la page Web InstallAnywhere Downloads.

Remarque – Les installateurs InstallAnywhere ne sont pas supportés par les versions bêta ni par les préversions (« access release ») de Java.

Nouvelles fonctions

Les nouvelles fonctionnalités d’InstallAnywhere permettent aux développeurs de builder des installations multiplate-forme de qualité. Les installateurs InstallAnywhere 8.0.1 sont supportés par Windows Vista et par les JVM Sun 1.6.x.

Corrections et améliorations

Les corrections et améliorations suivantes ont été apportées depuis la version 8.0.0 d’InstallAnywhere.

  • IOA-000030413: L’action « Set System Environment Variable / Définir une variable d’environnement système » est supportée par Windows Vista.
  • IOA-000027560: Cette nouvelle version permet de développer des projets pour HP-UX qui buildent des installateurs Web et des distributions sur CD.
  • IOA-000057176: Amélioration de la sécurité du script d’installation.
  • IOA-000029919: Un problème d’interface (GUI) qui empêchait l’action « Set Environment Variable / Définir une variable d’environnement » a été résolu.
  • IOA-000029119: Dans cette nouvelle version, la variable $WIN_COMMON_DESKTOP$ prend la valeur correcte pour les raccourcis Windows Vista.
  • IOC-000057327: Les règles ajoutées dans les Magic Folders fonctionnent correctement au runtime.
  • IOA-000027741: Le message de débogage « Command.run(): process completed before monitors could start / Le processus Command.run(): se termine avant le démarrage des moniteurs » ne s’affiche plus sur la console.
  • IOC-000058165: Les installateurs qui appliquent la règle « Check Platform / Vérifier la plate-forme » avec Windows Vista comme seule plate-forme cible s’installent correctement sous Windows Vista.

Lancer InstallAnywhere

Suivez les instructions correspondant au système d’exploitation sous lequel vous exécutez InstallAnywhere.

  • Pour lancer InstallAnywhere sous Windows :
    Dans le menu Démarrer, choisissez Programmes*Macrovision, puis InstallAnywhere 8 Enterprise ou InstallAnywhere 8 Standard. Vous pouvez également double-cliquer sur le fichier InstallAnywhere.exe, (par défaut, dans C:\Program Files\Macrovision\InstallAnywhere 8 Enterprise ou dans C:\Program Files\Macrovision\InstallAnywhere 8 Standard).
  • Pour lancer InstallAnywhere sous Mac :
    Double-cliquez sur InstallAnywhere (dossier Applications/Macrovision/InstallAnywhere 8 Enterprise ou Applications/Macrovision/InstallAnywhere 8 Standard). Vous pouvez également double-cliquer sur l’alias InstallAnywhere (créé par défaut dans le Dock).
  • Pour lancer InstallAnywhere sous UNIX (dont Solaris, Linux, HP-UX, AIX) :
    Entrez InstallAnywhere au niveau du dossier Macrovision/InstallAnywhere_8_Enterprise ou Macrovision/InstallAnywhere_8_Standard. Vous pouvez également exécuter le lien symbolique « InstallAnywhere » (installé par défaut dans le dossier racine).

Désinstaller InstallAnywhere

Suivez les instructions correspondant au système d’exploitation sous lequel vous exécutez InstallAnywhere.

  • Pour désinstaller InstallAnywhere sous Windows :
    Double-cliquez sur le raccourci Uninstall/Désinstaller (par défaut, dans C:\Program Files\Macrovision\InstallAnywhere 8 Enterprise ou dans C:\Program Files\Macrovision\InstallAnywhere 8 Standard).
  • Pour désinstaller InstallAnywhere sous Mac :
    Double-cliquez sur Uninstall/Désinstaller (dossier Applications/Macrovision/InstallAnywhere 8 Enterprise ou Applications/Macrovision/InstallAnywhere 8 Standard).
  • Pour désinstaller InstallAnywhere sous UNIX (dont Solaris, Linux, HP-UX et AIX) :
    Entrez Uninstall au niveau du dossier Macrovision/InstallAnywhere_8_Enterprise ou Macrovision/InstallAnywhere_8_Standard.
