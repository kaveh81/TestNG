概要

InstallAnywhere は、世界中のマルチプラットフォーム インストール開発者によって使用されているソリューションです。そのパワフルながらも使いやすい機能のおかげで、プロフェッショナルなマルチプラットフォームのインストーラを短時間で作成できます。InstallAnywhere には Standard 版と Enterprise 版があり、それぞれのユーザーのニーズに合わせて選ぶことができます。InstallAnywhere を使用すると、以下のことが可能です。

  • 複数のプラットフォームおよび複数の言語に対応した 1 つのインストーラ プロジェクトを作成できます。
  • 任意のクライアントまたはサーバー プラットフォームにソフトウェアを展開できます。
  • プロジェクト ウィザードを使ってマルチプラットフォームのインストーラを短時間で構築できます。
  • ソフトウェアの展開を簡単にするために、必要なプラットフォーム固有の機能をすべて含めることができます。

システム要件

InstallAnywhere を実行するシステム (オーサリング環境)

  • 使用可能な 128 MB の RAM
  • 最低 8 ビットの表示色 (256色)
  • 1024 x 768 以上の解像度

InstallAnywhere は、以下に示すオペレーティング システムで実行できます。お使いのオペレーティング システムが最新バージョンであり、最新のパッチとサービス パックで完全に更新されていることを確認してください。

  • Windows Server 2003、XP、2000、NT
  • Mac OS X、Mac OS X Server
  • Solaris/SPARC
  • Linux/Intel:Red Hat、SuSE
  • HP-UX
  • AIX 5.2 以降

メモ:ローカライズされた他言語の InstallAnywhere Eenterprise 版 (英語、フランス語、ドイツ語、および日本語の開発者用) は、Windowsプラットフォームでのみ使用できます。任意のプラットフォームまたは言語 (29 言語)のインストーラを、任意のプラットフォームで構築できます。

ターゲット システム (インストーラ環境)

  • 使用可能な 64 MB の RAM
  • 最低 8 ビットの表示色 (256色)
  • 640 X 480以上の画面解像度

インストーラは、以下のオペレーティング システムの任意のバージョンで実行できます。ただし、Java をサポートしているものに限ります。

  • Windows Server XP、2003 (Itanium 2、x86、AMD-64)
  • Windows Vista、2000、NT、Me、98
  • Linux/Intel:Red Hat、Red Hat Enterprise Server、UnitedLinux、TurboLinux、SUSE、その他
  • PowerPC、AMD-64、および Itanium 2 対応の Linux
  • Mac OS X (PPCとIntel)
  • Solaris SPARC、AMD-64、Intel
  • HP-UX (Itanium 2 と PA-RISC)
  • Tru64、FreeBSD
  • AIX
  • z/OS
  • NetWare
  • その他の Unix プラットホーム

メモ: ベータ版または早期アクセス版リリースでは、InstallAnywhere のインストーラを使用できません。

サポートされている Java VM

  • Sun:1.3.x、1.4.x、1.5.x、1.6.x
  • IBM:1.3.x、1.4.x、1.5.x
  • Apple:1.3.x、1.4.x、1.5.x
  • HP:1.3.x、1.4.x、1.5.x

InstallAnywhere のインストーラには、すべてのプラットホームに対してデフォルトで Java 1.5 がバンドルされます。インストーラに Java VM をバンドルすることで、ターゲット システムがインストーラとアプリケーションの両方の最低要件を満たすようにすることができます。これらのバンドル可能な VM パックをダウンロードするには、「InstallAnywhere Downloads (InstallAnywhere ダウンロード)」をご覧ください。

メモ: ベータ版または初期にリリースされた Java では、InstallAnywhere のインストーラを使用できません。

新機能

InstallAnywhere は、マルチプラットフォーム インストーラをビルドするために開発者が必要とする機能を提供してきました。InstallAnywhere 8.0.1 のインストーラは、Windows Vista システムや Sun 1.6.x JVM でもサポートされています。

バグの修正

InstallAnywhere 8.0.0 のリリース以降、以下の修正および強化が行われました。

  • IOA-000030413: Windows Vista で「システム環境変数を設定」クションが機能するようになりました。
  • IOA-000027560: HP-UX 用の CD-ROM 配布と Web インストーラの両方をビルドするプロジェクトが失敗しなくなりました。
  • IOA-000057176: インストール スクリプトのセキュリティ上の弱点がこの修正プログラムで強化されました。
  • IOA-000029919: 「システム環境変数を設定」クションに関連する GUI 問題が解決されました。
  • IOA-000029119: Windows Vista システムで、$WIN_COMMON_DESKTOP$ がショートカットに対して正しく解決されるようになりました。
  • IOC-000057327: MagicFolders に追加された規則が再びランタイムで機能するようになりました。
  • IOA-000027741: コンソールに、「モニタが起動する前に Command.run(): プロセスが完了しました」という旨のデバッグ メッセージが表示されなくなりました。
  • IOC-000058165: 唯一のプラットフォームとして Windows Vista に対して [プラットフォームのチェック] 規則を使用するインストーラが、Windows Vista のコンピュータに正しくインストールされるようになりました。

情報

フィードバックを送信する

お客様からのフィードバックは、お客様のニーズに応える機能を InstallAnywhere に搭載していく上で非常に重要な役割を果たします。InstallAnywhere 8.0.1 の機能の多くは、お客様、つまり InstallAnywhere 開発者から弊社のエンジニア チームまたはサポート チームに提供されたフィードバックや提案が基になっています。フィードバックを送信するには、Product Feedback (製品のフィードバック)にアクセスしてください。

InstallAnywhere を開く

InstallAnywhere を実行しているオペレーティング システム応じて、以下の手順で操作してください。

  • Windows で InstallAnywhere を開く場合:
    Windows の [スタート] メニューから [プログラム]、[Macrovision] の順にポイントし、[InstallAnywhere 8 Enterprise] または [InstallAnywhere 8 Standard] をクリックします。InstallAnywhere.exe をダブルクリックしても開くことができます。このファイルは、デフォルトで C:\Program Files\Macrovision\InstallAnywhere 8 Enterprise または C:\Program Files\Macrovision\InstallAnywhere 8 Standard にあります。
  • Mac で InstallAnywhere を開く場合:
    Applications/Macrovision/InstallAnywhere 8 Enterprise フォルダ内または Applications/Macrovision/InstallAnywhere 8 Standard フォルダ内で InstallAnywhere をダブルクリックします。デフォルトでドックに作成される InstallAnywhere エイリアスをダブルクリックしても開くことができます。
  • Solaris、Linux、HP-UX、AIX などの UNIX で InstallAnywhere を開く場合:
    Macrovision/InstallAnywhere_8_Enterprise フォルダまたは Macrovision/InstallAnywhere_8_Standard フォルダから「InstallAnywhere」と入力します。デフォルトでホーム フォルダにインストールされる InstallAnywhere シンボリック リンクを実行してもかまいません。

InstallAnywhere をアンインストールする

InstallAnywhere を実行しているオペレーティング システム応じて、以下の手順で操作してください。

  • Windows から InstallAnywhere をアンインストールする場合:
    Uninstall ショートカットをダブルクリックします。このショートカットは、デフォルトで C:\Program Files\Macrovision\InstallAnywhere 8 Enterprise または C:\Program Files\Macrovision\InstallAnywhere 8 Standard にあります。
  • Mac から InstallAnywhere をアンインストールする場合:
    Applications/Macrovision/InstallAnywhere 8 Enterprise フォルダ内または Applications/Macrovision/InstallAnywhere 8 Standard フォルダ内で [アンインストール] をダブルクリックします。
  • Solaris、Linux、HP-UX、AIX などの UNIX から InstallAnywhere をアンインストールする場合:
    Macrovision/InstallAnywhere_8_Enterprise フォルダまたは Macrovision/InstallAnywhere_8_Standard フォルダから「Uninstall」と入力します。
