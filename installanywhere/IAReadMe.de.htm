Einführung

InstallAnywhere ist weltweit die bevorzugte Lösung von Entwicklern für Multiplattform-Installationen. Es bietet die ideale Kombination aus Bedienerfreundlichkeit und leistungsstarken Funktionen zum Erstellen von professionellen Multiplattform-Installationen in wenigen Minuten. InstallAnywhere ist als Standard- und Enterprise-Edition verfügbar, um Ihre spezifischen Anforderungen zu erfüllen. InstallAnywhere bietet folgende Hauptfunktionen:

  • Erstellung eines einzigen Installer-Projekts für mehrere Plattformen und mehrere Sprachen
  • Installation von Software auf beliebigen Client- oder Serverplattformen
  • Erstellung einer Multiplattform-Installation in wenigen Minuten mit dem Projektassistenten
  • Bereitstellung aller plattformspezifischen Funktionen, die Sie zur Vereinfachung Ihrer Software-Installation benötigen

Systemvoraussetzungen

Systeme, auf denen InstallAnywhere installiert ist (Autorenumgebung)

  • 128 MB freier Arbeitsspeicher
  • Farbtiefe mind. 8 Bit (256 Farben)
  • Auflösung mindestens 1024 x 768 Pixel

InstallAnywhere läuft unter den neuesten Versionen der folgenden Betriebssysteme mit den neuesten Aktualisierungen, Patches und Service-Packs:

  • Windows Server 2003, XP, 2000, NT
  • Mac OS X, Mac OS X Server
  • Solaris/SPARC
  • Linux/Intel: Red Hat, SuSE
  • HP-UX
  • AIX 5.2 oder höher

HINWEIS: Lokalisierte Versionen von InstallAnywhere Enterprise Edition (für Entwickler, auf Deutsch, Englisch, Französisch und Japanisch) sind nur für Windows erhältlich. Installer können aber auf jeder Plattform für jede beliebige andere Plattform oder Sprache erstellt werden. Lokalisierung für 29 Sprachen bereits enthalten.

Zielsysteme (Installer-Umgebung)

  • 64 MB freier Arbeitsspeicher
  • Farbtiefe mind. 8 Bit (256 Farben)
  • Bildschirmauflösung mindestens 640 x 480 Pixel

Installer funktionieren unter jeder Version der folgenden Betriebssysteme, sofern das Betriebssystem Java unterstützt.

  • Windows Server XP, 2003 (Itanium 2, x86, AMD-64)
  • Windows Vista, 2000, NT, Me, 98
  • Linux/Intel: Red Hat, Red Hat Enterprise Server, UnitedLinux, TurboLinux, SuSE und andere
  • Linux für PowerPC, AMD-64 und Itanium 2
  • Hinweise zu Mac OS X (PPC und Intel)
  • Solaris SPARC, AMD-64, Intel
  • HP-UX (Itanium 2 und PA-RISC)
  • Tru64, FreeBSD
  • AIX
  • z/OS
  • NetWare
  • Andere UNIX-Plattformen

HINWEIS: InstallAnywhere-Installer werden nicht auf Beta- und Frühversionen unterstützt.

Unterstützte Java Virtual Machines

  • Sun: 1.3.x, 1.4.x, 1.5.x, 1.6.x
  • IBM: 1.3.x, 1.4.x, 1.5.x
  • Apple: 1.3.x, 1.4.x, 1.5.x
  • HP: 1.3.x, 1.4.x, 1.5.x

Mit dem InstallAnywhere-Installer ist auf allen Plattformen standardmäßig Java 1.5 gebündelt. Es kann aber auch eine beliebige andere Java Virtual Machine gebündelt werden, um zu gewährleisten, dass das Zielsystem die Mindestvoraussetzungen für den Installer und Ihre Anwendungen erfüllt. Diese gebündelten VM-Pakete können Sie unter InstallAnywhere Downloads herunterladen.

HINWEIS: InstallAnywhere-Installer werden nicht auf Beta- und Frühversionen von Java unterstützt.

Neue Funktionen

InstallAnywhere liefert weiterhin alle Funktionen, die Entwickler zur Erstellung erfolgreicher Multiplattform-Installationen benötigen. InstallAnywhere 8.0.1-Installer werden jetzt unter Windows Vista und Sun 1.6.x JVMs unterstützt.

Behobene Fehler

Seit InstallAnywhere 8.0.0 wurden die folgenden Fehlerbeseitigungen und Verbesserungen vorgenommen.

  • IOA-000030413: Die Aktion "Variable für Systemumgebung einstellen" funktioniert jetzt unter Windows Vista.
  • IOA-000027560: Projekte, die sowohl Web- als auch CD-ROM-Installer für HP-UX erzeugen, funktionieren jetzt.
  • IOA-000057176: In diesem Hotfix-Patch wurde eine Sicherheitslücke im Installationsscript geschlossen.
  • IOA-000029919: Ein Problem in der GUI im Zusammenhang mit der Aktion 'SetEnvironmentVariable' wurde behoben.
  • IOA-000029119: $WIN_COMMON_DESKTOP$ wird jetzt für Verknüpfungen auf Windows Vista ordnungsgemäß aufgelöst.
  • IOC-000057327: Regeln für Sonderordner funktionieren jetzt wieder zur Laufzeit.
  • IOA-000027741: Die Debugging-Meldung "Command.run(): process completed before monitors could start" ("Prozess vor Start der Überwachung abgeschlossen") wird nicht mehr in der Konsole angezeigt.
  • IOC-000058165: Auf Computern, auf denen als einziges Betriebssystem Windows Vista installiert ist, funktionieren Installer, die die Regel "Plattform prüfen" verwenden, jetzt ordnungsgemäß.

InstallAnywhere öffnen

Gehen Sie nach der Anleitung für das Betriebssystem vor, auf dem Sie InstallAnywhere ausführen.

  • So öffnen Sie InstallAnywhere unter Windows:
    Zeigen Sie im Start-Menü von Windows auf Programme und dann auf Macrovision. Klicken Sie auf InstallAnywhere 8 Enterprise bzw. InstallAnywhere 8 Standard. Sie können auch auf die Datei InstallAnywhere.exe doppelklicken, die sich standardmäßig im Verzeichnis C:\Program Files\Macrovision\InstallAnywhere 8 Enterprise bzw. C:\Program Files\Macrovision\InstallAnywhere 8 Standard. befindet.
  • So öffnen Sie InstallAnywhere auf dem Macintosh:
    Doppelklicken Sie im Ordner Applications/Macrovision/InstallAnywhere 8 Enterprise bzw. Applications/Macrovision/InstallAnywhere 8 Standard auf InstallAnywhere. Sie können auch auf den Alias "InstallAnywhere" doppelklicken, der standardmäßig in Ihrem Dock erstellt wird.
  • So öffnen Sie InstallAnywhere unter UNIX, einschließlich Solaris, Linux, HP-UX, AIX u. a.:
    Geben Sie InstallAnywhere im Ordner Macrovision/InstallAnywhere_8_Enterprise bzw. Macrovision/InstallAnywhere_8_Standard ein. Sie können auch die Symbolverknüpfung für InstallAnywhere ausführen, die standardmäßig in Ihrem Startordner installiert wird.

InstallAnywhere deinstallieren

Gehen Sie nach der Anleitung für das Betriebssystem vor, auf dem Sie InstallAnywhere ausführen.

  • So deinstallieren Sie InstallAnywhere unter Windows:
    Doppelklicken Sie auf die Verknüpfung Deinstallieren (Uninstall), die sich standardmäßig im Ordner C:\Program Files\Macrovision\InstallAnywhere 8 Enterprise bzw. C:\Program Files\Macrovision\InstallAnywhere 8 Standard befindet.
  • So deinstallieren Sie InstallAnywhere auf dem Macintosh:
    Doppelklicken Sie im Ordner Applications/Macrovision/InstallAnywhere 8 Enterprise bzw. Applications/Macrovision/InstallAnywhere 8 Standard auf Deinstallieren (Uninstall).
  • So deinstallieren Sie InstallAnywhere unter UNIX, einschließlich Solaris, Linux, HP-UX, AIX u. a.:
    Geben Sie Deinstallieren (Uninstall) im Ordner Macrovision/InstallAnywhere_8_Enterprise bzw. Macrovision/InstallAnywhere_8_Standard ein.
